## 📌 Parte 1 - Hello 3D

### Descrição

Implementação inicial do ambiente gráfico 3D utilizando OpenGL com Python. O programa exibe um **triângulo colorido estático** no centro da janela, com os vértices nas cores vermelho, verde e azul (RGB).

### Requisitos Atendidos

✅ Criar repositório no GitHub  
✅ Configurar ambiente virtual Python  
✅ Instalar dependências (GLFW, PyOpenGL, NumPy)  
✅ Implementar Hello3D com OpenGL 3.3+  
✅ Título da janela: "Ola 3D - Juliane Correa"  
✅ Criar arquivo RESULT.md com print da execução  

### Tecnologias Utilizadas

| Ferramenta | Versão | Finalidade |
|------------|--------|------------|
| Python | 3.x | Linguagem principal |
| GLFW | 2.5.5 | Criação de janela e entrada |
| PyOpenGL | 3.1.7 | Bindings OpenGL para Python |
